# 🎓 Student Information System

##  Project Overview

This is a web-based Student Information System developed using HTML, CSS, and PostgreSQL. It provides role-based dashboards for Students, Teachers, and Admin.

## Features

*  User Authentication (Login/Signup)
*  Student Dashboard (Profile, Courses, Results, Attendance)
*  Teacher Dashboard (Manage Marks, Attendance, Courses)
*  Admin Dashboard (Manage Users & Courses)
*  Result Management System
*  Attendance Tracking
*  Notifications System

##  Technologies Used

* HTML
* CSS
* PostgreSQL

* database.sql
